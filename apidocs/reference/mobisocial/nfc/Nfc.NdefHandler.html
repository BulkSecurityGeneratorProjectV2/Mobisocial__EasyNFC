A priority value used to determine the order in which ndef handlers are issued. Lower priority handlers are issued first. The default priority is 50.

public abstract int handleNdef (NdefMessage[] ndefMessages)

Callback issued after an NFC tag is read or an NDEF message is received from a remote device. This method is executed off the main thread, so be careful when updating UI elements as a result of this callback.

Returns
  • NDEF_CONSUMED to indicate this handler has consumed the given message, or NDEF_PROPAGATE to pass on to the next handler.
