EasyNFC is a library for NFC interactions on Android. Features:
  * Graceful degredation on platforms lacking NFC APIs
  * Suport "virtual NFC", allowing two devices to communicate via NDEF exchange if only one device has an active NFC radio
  * Helps developers set up long-lasting Bluetooth connections across devices:
       http://mobisocial.github.com/EasyNFC/apidocs/reference/mobisocial/nfc/addon/BluetoothConnector.html

More documentation is available at:
    http://mobisocial.github.com/EasyNFC/apidocs/reference/mobisocial/nfc/Nfc.html
